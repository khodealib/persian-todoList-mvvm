# Tools and Techs:
* Architecture: **MVVM**
* MultiThreading: **RxJava2**
* UI: **Material Design**
* Android Architecture Components: **LiveData, ViewModels**
* We use **Repository** in model layer
* In presentation layer we use **LiveData** to bind data to view
* Language **Persian**
